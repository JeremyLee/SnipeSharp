# Get-SnipeRequest

## SYNOPSIS
Gets Snipe IT requests.

## SYNTAX

```
Get-SnipeRequest [<CommonParameters>]
```

## DESCRIPTION
The Get-Request cmdlet gets all requests for the current account.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-Request
```

Retrieve all requests for the current account.
